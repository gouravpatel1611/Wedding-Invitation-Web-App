# 💍 Shubh Vivaah - Digital Wedding Invitation

A modern and interactive **wedding invitation web application** built using React.
This project transforms traditional wedding cards into a beautiful digital experience that can be shared easily via a link.

---

## 🌐 Live Preview

*https://gouravwedding.vercel.app/*

---

## ✨ Features

* 💌 Personalized invitation using dynamic URL (`?name=Guest`)
* 🎨 Elegant wedding UI with Hindi content
* 🖼️ Full-screen background with couple photo
* 🎉 Opening invitation screen (card style)
* 📜 Wedding events (before, during, after)
* 📍 Venue location with map integration
* 📱 Fully responsive (mobile friendly)
* 📞 Contact section for guests

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 🎨 CSS / Inline Styling
* 🌐 HTML5

```

```

```

### 3. Run the project

```
